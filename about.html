    About this site

    Motivaions

    I created this site as a way to help my children learn more about computers and how they work. I also decided to create something from scratch rather than using some of the already available resources so that it would be a fun project for me as well. The content on this site is now, and will always be, free for any use. Contributions are welcome, provided you are agreeable to the same terms.

    Shoulders being stood upon

    This website is built using several other wonderful projects.

    Skulpt

    Skulpt is what allows the code presented and modified on this web site to be run entirely within your web browser.

    CodeMirror

    CodeMirror is what is styling the boxes of code that are displayed on this site. I provides syntax highlighting and other nice display features.

    Bootstrap

    Bootstrap is used for the web design, styling, layout, and icons.

    Jinja2

    The HTML pages that constitute this site are built, in part, using the Jinja2 templating language.

    Python

    All of the code on this site is written in a programming language called Python. Python is, in my opinion, an excellent language for teaching the fundimental concepts of computer software.
